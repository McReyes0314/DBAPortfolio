Process

In this activity, we have been tasked with designing a database table and identifying the most appropriate SQL commands. Below are the detailed instructions given to us:

Question 1

Design a database table for managing library books. include the following requirements:

  • Columns: book_id (Primary Key, Integer), title (VARCHAR(100)), author (VARCHAR(50)), published_year (YEAR), genre (VARCHAR(30)), copies_available (Integer).
  • Write the SQL statement to creare the table.
  • Write an SQL query to insert the following book details into the table:
    Book ID: 101, Title: "SQL Fundamentals", Author: "John Doe", Published Year: 2022, Genre: "Technology", Copies Available: 5.
  • Write an SQL query to update the copies_available for the book with ID 101 to 7.
  • Write an SQL query to delete all books published before 2000.

Question 2

For each scenario below, identify the most appropriate SQL command (CREATE, ALTER, DROP, SELECT, INSERT, UPDATE, DELETE, GRANT, REVOKE) and provide a brief justification for your choice.

  • You need to create a new table in the database to store employee records
  • A column in the "student" table needs to hava a default value updated
  • You want to delete all records from the "products" table but keep the table structure.
  • A user needs permission to view and query the the "sales" table
  • You need to remove the "inventory" table entirely from the database

Learnings

Throughout the activity, I learned the following aspects

  • I learned how to create a database and design tables with specified columns and constraints.
  • I learned to identify the appropriate SQL command for specific questions being asked.
